# Sniffer_redes
Proyecto sniffer de programación de aplicaciones de redes, versión beta y versiones posteriores. 
Podría haber cambios, aún está en desarrollo. 
Para que funcione hace falta abrir VS Code como administrador, Windows es medio exigente en ese aspecto. 
Una vez abierto se ejecuta con Run & Debug, enseguida se abre la GUI hecha con Tkinter, puedes usar la pestaña superior derecha para filtrar diferentes protocolos, o dejar por default todos para que intercepte de todo un poco. 
Se le da clic a iniciar, una vez iniciado verás en pantalla como los paquetes son interceptados, puedes hacer pruebas con un ping 8.8.8.8 desde la terminal y filtrado por ICMP para visualizar más facilmente con esta prueba rápida. 
Usa el botón detener cuándo desees detener la intercepción de paquetes. 
Con el botón limpiar, la interfaz queda limpia y lista para usarse de nuevo.
